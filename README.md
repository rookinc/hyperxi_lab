# HyperXi Spectral Transport Lab

Research scaffold for studying discrete transport on the lifted flag space
of the {5,3,4} dodecahedral lattice.

Core components:

- Combinatorial dodecahedron incidence model
- 120 lifted flags
- Coxeter-style generators (S, F, V)
- Local transport operator
- Cycle enumeration experiments

Primary scripts:

scripts/compute_spectrum.py
scripts/enumerate_cycles.py
