\section{Notation Summary}

We summarize the primary objects used throughout the paper.

\begin{itemize}
\item $G_{60}$: chamber graph
\item $G_{30}$: intermediate quotient
\item $G_{15}$: deepest quotient, isomorphic to $L(\mathrm{Petersen})$
\item $\varepsilon$: transport invariant on chambers
\item $\sigma$: induced edge sign function on $G_{15}$
\item $A$: adjacency matrix of $G_{15}$
\item $M$: sector incidence matrix
\end{itemize}

\item \textbf{Cubic Thalean representation}: the spherical embedding of
$G_{15} \cong L(\mathrm{Petersen})$ induced by the polynomial
$p(x) = x^3 + 2x^2 + 2$ via the relation $M M^T = p(A)$
