\section{Notation Summary}

We collect the main objects used throughout the paper.

\begin{itemize}
\item $\Gsixty$: the $60$-vertex chamber graph obtained from dodecahedral
flag transport.

\item $\Gthirty$: the intermediate $2$-fold quotient of $\Gsixty$.

\item $\Gfifteen$: the $15$-vertex core graph, identified with
$L(\mathrm{Petersen})$.

\item $\varepsilon : E(\Gfifteen)\to\ZZtwo$: the transport cocycle.

\item $\omega(C)$: holonomy of a cycle $C\subset \Gfifteen$.

\item $H^1(\Gfifteen,\ZZtwo)$: mod-$2$ cohomology of the core graph.
\end{itemize}
