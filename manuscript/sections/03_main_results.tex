\section{Main Results}

We summarize the principal results of this paper.

\begin{theorem}
The chamber graph $\Gsixty$ is connected, $4$-regular, and admits
successive $2$-fold quotients
\[
\Gsixty \to \Gthirty \to \Gfifteen,
\]
with $\Gfifteen \cong L(\mathrm{Petersen})$.
\end{theorem}

\begin{theorem}
The lift $\Gthirty \to \Gfifteen$ defines a canonical cocycle
\[
\varepsilon : E(\Gfifteen)\to \ZZtwo,
\]
recording sheet holonomy.
\end{theorem}

\begin{theorem}
The cocycle $\varepsilon$ represents a nontrivial class in
$H^1(\Gfifteen,\ZZtwo)$ and is not determined by any local rule.
\end{theorem}
