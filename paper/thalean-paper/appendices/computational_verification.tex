\section{Computational Verification}

The structural properties of the graph $G_{60}$ were verified
computationally using scripts in the \texttt{hyperxi\_lab} repository.

The full adjacency structure of $G_{60}$, the explicit involutions
generating the $V_4$ action, and the orbit decomposition used in the
analysis are provided in the supplementary file

\texttt{export_thalean_graph_definition.json}.
